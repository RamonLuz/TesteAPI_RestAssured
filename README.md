# TesteAPI_RestAssured# 🧪 API Tests com Rest Assured

Projeto simples de automação de testes de API utilizando **Rest Assured**, **JUnit**, **Java** e **Cucumber**.

---

## 📌 Sobre o projeto

Este projeto foi criado para demonstrar testes básicos de API REST, incluindo:

* Validação de status code
* Validação de payload (JSON)
* Testes de métodos HTTP (GET, POST)

---

## 🚀 Tecnologias utilizadas

* Java 11+
* Maven
* Rest Assured
* JUnit 4

---

## 📂 Estrutura do projeto

```
src
 └── test
      └── java
           
```

---

## 🔍 Boas práticas aplicadas

* Uso de `given / when / then`
* Separação de testes por responsabilidade
* Validações claras e objetivas
